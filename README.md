# My Music App (Flutter Experiment)

This is an open-source educational project developed to explore application architecture and project management through "vibe coding" (AI-assisted development).

## 🧠 The "Vibe Coding" Journey
I started this project with **zero prior knowledge of Flutter or Dart**. The entire development process is driven by AI assistance. My primary goal isn't just to write code, but to learn:
* How software projects are structured and managed.
* How to communicate effectively with AI to build a full application.
* The overall lifecycle of client-side application development.

## 🚧 Purpose & Disclaimer
This project is for **educational and research purposes only**. 
* It is NOT intended for commercial use.
* It does not host any copyrighted content.
* It acts as a client-side interface demonstration.

## 🛠 Tech Stack
* **Framework:** Flutter (Dart)
* **Audio Engine:** just_audio
* **State Management:** Provider
* **Local Storage:** Hive
* **Development Approach:** AI-Assisted "Vibe Coding"

## ⚠️ Note on Usage
This application is a Proof of Concept (PoC). No APK binaries are distributed within this repository.
